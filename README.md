# Master's Thesis: Are Student-Led Protests a Force for Democracy?

Replication code for my master's thesis at the University of Oslo, May 2026.

## Files

- `01_descriptive_statistics.R` — Descriptive figures and tables
- `02_main_analysis.R` — Main models and robustness checks

## Data

The analysis uses the Opposition Movements and Groups (OMG), Varieties of Demcoracy (V-Dem), World Bank indicators (WDI), and Our World in Data (OWID) datasets. Data files are not included. 

## R version

R 4.5.1 (2025-06-13). Required packages are loaded at the top of each script.
